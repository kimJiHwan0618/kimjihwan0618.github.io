## 로컬 실행 방법

루비가 설치되어 있는 경우 다음 단계를 따르세요:

1. 저장소를 클론합니다.
    ```shell
    git clone https://github.com/kimjihwan0618/kimjihwan0618.github.io.git
    cd your-repo-name
    ```

2. 의존성을 업데이트하고 설치합니다.
    ```shell
    bundle update
    bundle install
    ```

3. Jekyll 서버를 실행합니다.
    ```shell
    bundle exec jekyll serve
    ```
